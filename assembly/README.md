# Assembly Guide

How to build the chair, step by step.

> **The canonical assembly steps live in the bundled PDF:**
> [`../docs/assembly-guide.pdf`](../docs/assembly-guide.pdf) (4 steps, ~45 min).
>
> This file is a short companion summary. Photos and a build video are
> **to be updated** — `photos/step-XX.jpg` filenames are placeholders.

## Before You Start

1. Read the PDF assembly guide end-to-end
2. Lay out all parts (`bom/parts.csv`)
3. Tools needed: the included Allen wrench, a Phillips screwdriver, a rubber mallet

## Time

~45 minutes for a first build.

## Steps (summary)

### Step 1 — Base + casters + gas cylinder

Flip the 5-arm base upside down and press the 5 casters into their holes
(rubber mallet, no glue). Flip the base over and drop the class-4 gas
cylinder into the centre socket — friction-fit.

### Step 2 — Seat + back

With the seat upside-down on a stable edge, attach the back to the seat
using the **4× M6×25 Allen screws** and **4× tapping screws** (8 screws total).
Use the included Allen wrench for the Allens; a Phillips driver for the
tappers.

### Step 3 — Headrest

Align the headrest into the back-frame slot and secure with **2 screws**
using the Allen wrench.

### Step 4 — Drop chair onto base

Lift the assembled chair (B) and lower it onto the gas-cylinder taper (A).
The chair's weight locks it.

## Adjustments

After assembly, the chair has six adjustments — see the PDF:

1. Seat depth (front lever)
2. Tilt / recline tension (crank under seat)
3. Seat height (right-side flip-up lever)
4. Sliding armrest (grasp arm pads, slide in/out)
5. Armrest height (button under arm pad)
6. Back tilt lock (left-side flip-down lever)

## Troubleshooting

- Gas cylinder doesn't lock → make sure the taper is clean and dry, then sit
  on the chair to seat it.
- Back wobbles → re-check that all 8 screws in Step 2 are tight.
